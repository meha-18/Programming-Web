Introduction :

Python is a high-level, interpreted, general-purpose programming language. Its design philosophy emphasizes code readability with the use of significant indentation.

Why Python?

It is versatile, easy to learn, and widely used in Web Development, Data Science, and Cybersecurity.

Key Features :

  • Interpreted: Python code is executed line by line.

  • Dynamically Typed: You don't need to declare variable types.

  • Large Standard Library: Offers many modules and functions for various tasks.
  • Cross-platform: Runs on Windows, Linux, and macOS.

Data Types Table :

Type Example Description
int x = 10 Integers (whole numbers without decimals).
float y = 10.5 Floating point numbers (contain decimals).
complex Z = 3j Complex numbers (used in mathematical calculations).
str name = "Maha" Strings (sequences of characters).
list L = [11, 18, "m"] Ordered, mutable collection (can be changed).
tuple T = (11, 18, 4) Ordered, immutable collection (cannot be changed).
dict {"id" : 104} Dictionary (Key-Value pairs).
set {"apple", "orange"} Unordered collection of unique items.
bool True / False Logical values representing truth or falsehood.
